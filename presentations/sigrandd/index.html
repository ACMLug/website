LUG@UIUC - A .signature Randomizing Daemon Using FIFOs

To illustrate how to write a daemon and how to interact with FIFOs, the following sample code and documentation was written by Steven Engelhardt:

  • The Makefile to generate the daemon and documentation
  • The C source code to the daemon
  • The SGML source code to the documentation

In addition, since you may not have SGMLtools installed and thus cannot compile the documentation, we provide documentation in the following formats:

  • HTML
  • Postscript

Note: Other formats, including PDF, DVI, plain ASCII, etc. can be generated from the SGML source code.
