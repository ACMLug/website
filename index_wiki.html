What is Linux?

Linux is a freely-distributable implementation of POSIX standards (or it at least aims to implement them). The project was started from scratch by Linus Torvalds while he was a Computer Science student at the University of Helsinki in Finland. It has since grown to be a collobarative effort among thousands of people across the Internet.

The main kernel page is kernel.org. Debian GNU/Linux, SUSE Linux, Gentoo Linux, and many others combine all the software needed into one easy to find package.

Included with Linux you can find:

  • Open Office, a open source office productivity tool compatable with Microsoft Office.
  • A variety of really cool open source software. Web browsing, email programs, AIM/ICQ/MSN/Yahoo clients, word processors, spreadsheets, etc.
  • A complete multitasking operating system that uses virtual memory and shared libraries. It includes full TCP/IP networking support, including FTP, Telnet, NFS, PPP, SLIP, etc.
  • Compilers, communication programs, games, editors, etc.
  • XFree86, an implementation of the X Window System, which includes many useful graphical tools.
Some Linux Myths
With so many people unfamiliar with Linux, misconceptions about what Linux is and is not have become widespread, to name a few:
  • Linux is hard to install! - Linux is extremely easy to install as a machine's primary operating system and so that it dual boots with Windows. It is slightly harder if your Windows partition is formatted as NTFS.
  • Linux is too hard to use - Linux is a UNIX-like operating system. In short, Linux is different, not difficult. Usability studies have shown that it is comparable to Windows in terms of the GUI. The underlying power underneath Linux is the command line, however. If you choose to run Linux these days you don't even have to know how to use the command line, however it is one of the most powerful parts of Linux.
  • Linux doesn't ever crash - well, okay, this one's true. Linux is very difficult to crash, a feature that stems from the rigorous peer review generated by open source development and the permissions based file systems dictated by the POSIX standards.
