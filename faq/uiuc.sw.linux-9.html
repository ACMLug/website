This document was originally created by Steven Engelhardt (sengelha@uiuc.edu), with help from Andrew Ho, Mark Roth, Todd Ostermeier, Jay Tamboli, David Terrell, and others.

9.2 How was this document created?

This document was originally written in SGML. Using the Debian sgmltools package, it is converted into the various formats for distribution, which includes HTML, ASCII, and PostScript.
