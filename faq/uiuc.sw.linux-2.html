2. Installation and Booting

2.1 How can I dual boot Windows 9x and Linux?

Please see the Linux+Win95 HOWTO at http://metalab.unc.edu/LDP/HOWTO/mini/Linux+Win95.html.

2.2 How can I dual boot Windows NT(tm) and Linux?

If you plan on using LILO as your primary bootloader, the instructions should be nearly exactly the same as in "How do I dual boot Windows 9x and Linux?" If you plan on using the NT bootloader, please see the Linux+NT-Loader HOWTO at http://metalab.unc.edu/LDP/HOWTO/mini/Linux+NT-Loader.html.
