ACM Vending is an adaptable, expandable interface for vending machines. Written in Python, ACM Vending provides a customizable user interface (using a templated HTML format), pluggable options for databases, telnet interface, and web frontend, all implemented in a single Python binary.

ACM Vending is currently powering ACM's Caffeine vending machine and authenticates users using their iCards and ACM's internal account information.
